1. 자료형
- 데이터의 형태를 의미한다.
- 정수, 실수로 이루어진 숫자 자료형, 문자열로 이루어진 문자열 자료형, 참 True, 거짓 False 만 값으로 가지는 불 자료형
- 문자열 자료형은 ''나 ""로 감싸야 한다.
  - 숫자 자료형 : 5, -10, 3.14, 1000
  - 문자열 자료형 : '풍선', "나비"
  - 불 자료형 : True, False

2. 변수
- 어떤 값을 저장하는 공간
- 변수에 값을 저장시 변수명 = 값 형식으로 작성한다.

3. 형변환
- 값의 자료형을 다른 형태로 바꾸는 것을 의미한다.
  - str() : 값을 문자열로 형변환
  - int() : 값을 정수형으로 형변환
  - flast() : 값을 실수형으로 형변환

4. 주석
- 코드에서 실행하지 않도록 처리하는것
- 한줄 주석 # 사용, 여러줄 주석은  """ """ 사용
